# Rezoria CL

Public repo for `REZORIA_CL.lua` and a small obfuscated Lua loader.

The loader downloads the current raw `REZORIA_CL.lua` from GitHub and executes it in OTClient/vBot.

## Install

Load `loader.lua` in vBot.

## Security Note

This repository is public, so `REZORIA_CL.lua` is public. The loader only hides the raw GitHub URL inside the loader; it is not a source-code protection boundary.
